# 💰 Credit Risk AI System

A machine learning web app that predicts loan default risk and explains every decision using SHAP — built with Streamlit, XGBoost, and Explainable AI techniques.

---

## ✨ Features

| Feature | Details |
|---|---|
| **3 ML Models** | Logistic Regression, Random Forest, XGBoost |
| **Single Prediction** | Real-time risk score + SHAP waterfall explanation |
| **Batch Prediction** | Upload CSV → get predictions + downloadable results |
| **Model Metrics Dashboard** | AUC-ROC, Avg Precision, CV scores side by side |
| **Global SHAP** | Summary bar plots + dependence plots across the dataset |
| **Secure Login** | Credentials via `st.secrets` — no hardcoded passwords |
| **Fast Reruns** | `@st.cache_resource` so models load once, not every click |

## 🚀 Quick Start

### 1. Clone & install
```bash
git clone https://github.com/yourname/credit-risk-ai.git
cd credit-risk-ai
pip install -r requirements.txt
```

### 2. Add the dataset
Download the Loan Default dataset from Kaggle and save it to:
```
data/Loan_default.csv
```

### 3. Set credentials
```bash
mkdir -p .streamlit
cat > .streamlit/secrets.toml << 'EOF'
[auth]
username = "admin"
password = "your_secure_password"
EOF
```

### 4. Train models
```bash
cd src
python train.py
```
This saves all three `.pkl` files and `models/metrics.json`.

### 5. Run the app
```bash
streamlit run app.py
```

## 📊 Dataset

- **Source:** [Loan Default Prediction — Kaggle](https://www.kaggle.com/)
- **Size:** ~255,000 applicants
- **Target:** `Default` (0 = no default, 1 = default)

| Model | AUC-ROC | Avg Precision |
|---|---|---|

| Random Forest | ~0.89 | ~0.62 |

Exact values are stored in `models/metrics.json` after training and displayed live in the app's **Model Metrics** tab.
